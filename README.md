# ◈ AI Notes Organizer

> AI-powered lecture notes organizer. Upload PDF, image or text → get instant study materials.

🔗 **Live Demo:** https://ai-notes-organizer.vercel.app

## ✨ Features
- 💬 AI Chat — answers from YOUR notes
- 📖 Study Guide — auto-structured notes  
- 🃏 Flashcards — flip cards + Anki export
- 🧠 Mind Map — interactive knowledge tree
- 🔢 Formula Sheet — auto-extracts equations
- 🔍 Knowledge Gaps — finds missing topics
- 📝 MCQ Quiz — test + results dashboard
- ↓ Download — full HTML study package

## 🛠️ Tech Stack
- Frontend: React.js + Framer Motion
- Backend: Node.js + Express
- AI: OpenRouter API (free tier)
- PDF: pdf-parse | OCR: Tesseract.js

## ⚡ Run Locally
```bash
# Clone
git clone https://github.com/kunalghuge048-pixel/ai-notes-organizer.git

# Backend
cd backend
npm install
# Create .env with your OpenRouter key
node server.js

# Frontend (new terminal)
cd frontend
npm install
npm start
```
